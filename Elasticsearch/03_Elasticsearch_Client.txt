### ELASTICSEARCH CLIENT ###
Elasticsearch menggunakan RESTful API sebagai cara berkomunikasinya.
Oleh karena itu, kita cukup menggunakan HTTP untuk berkomunikasi dengan Elasticsearch.
Karena menggunakan HTTP, secara otomatis lebih mudah ketika kita belajar Elasticsearch.

##Aplikasi HTTP Client 
- Postman 
  https://www.postman.com/ 

- Insomnia 
  https://insomnia.rest/
